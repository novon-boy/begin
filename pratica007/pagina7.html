Imagens flexíveis

Após criar imagens de tamanho de 1000x1000, 700x700, e 300x300, usaremos para criar imagens flexiveis para cada tipo de aparelho, usaremos esses tamanhos como exemplo, você que irá decidir como ficará o tamanho das imagens no seu site para cada aparelho. Enfim, aqui o código para por essas imagens em HTML:

        
            <picture>
                <source media="(max-width=750px)" srcset="imagempequena.png" type="image/png">
                <source media="(max=width=1050px)" srcset="imagemmedia.png" type="image/png">
                <img src="imagemgrande.png" alt="imagemgrande">
            </picture>
media="(max-width=)"
Define o máximo de pixel que a imagem continuará exibindo na tela, caso seja usado "min-width", será o contrário, este irá definir o minimo de pixels necessário para a imagem aparecer na tela
srcset=""
Imagem que será exibida se as condições do width forem cumprimdas
type=""
Tipo da imagem
img src="" ...
Imagem que será exibida como base, ou caso os outros dois sources não estiverem corretos, como por exemplo se a tela já estiver com 1200px

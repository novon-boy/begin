Audios

Para por audios no site podemos usar um simples, <audio src="" controls></audio> mas é muito simples e caso o navegador não suporte o áudio não terá outras opções, para isso usamos o seguinte código

        
            <audio preload="" controls autoplay loop>
                <source src="caminhoaudio.mp3" type="audio/mpeg">
                <source src="caminhodoaudio.ogg" type="audio/ogg">
                <source src="caminhoaudio.wav" type="audio/wav">
                <p>Mensagem de erro ao tentar colocar o audio no site <a href="caminhoaudio.mp3" download="audio.mp3" type:"audio/mpeg">mensagem para download</a></p>
            </audio>

preload=""

metadata
vai carregar as principais informações do arquivo(tamanho, tempo, informações de direitos, etc)
none
Não vai carregar absolutamente nada até que o usuário clique no botão play ou o script inicie a reprodução
auto
padrão, vai carregar o arquivo do áudio inteiro assim que a página for carregada, mesmo que o usuário nunca aperte o play

controls, autoplay e loop

controls
mostra os controles para controlar o áudio, botão de play, duração, etc
autoplay
reproduz o áudio assim que a aba é aberta, não funcionando hoje 13/03/2025
loop
Looping no áudio, toca o áudio repetidas vezes sem parar, a não ser que o usuário pause

source src="" type=""

source
Usado para dar várias opções de formatos de áudio caso algum não seja compátivel com o navegador
src=""
caminho do áudio
type=""
tipo do áudio

o parágrafo no final

criamos o paragrafo no final para exibir uma mensagem, no caso uma mensagem com um link para o download do audio em mp3, caso os nenhum source(nenhum áudio) seja compativel com o navegador

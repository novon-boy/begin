Formatação de textos com semântica

HTML é uma língua baseada na semântica, isso significa que ela foca no significado, ou o seja, o significado que o conteúdo quer passar, que difente do CSS, que foca em forma, no estilo do conteúdo

Exemplos de formatações

Negrito
Usamos a tag strong, para dar sentido de destaque, forte
Itálico
Usamos a tag em, para dar sentido de ẽnfase(emphasis)
Marca texto
Usamos a tag mark, que marca a frase ou palavra como um marca texto; é possível mudar a cor usando <mark style background-color: "cor"> para mudar a cor de um mark, para mudar a cor de todos, criamos um style no head, e criamos um seletor para o mark. Mas a mudança de cor é uso do CSS
Letras pequenas
Usamos a tag small, pois diferente da tag big, as letras pequenas podem ter significado, como por exemplo em documentos
Texto deletado
Usamos a tag del, que mostra que o texto deve ser lido mas não considerado
Texto inserido
Usamos a tag ins, para mostrar destaque ou mostrar que o texto foi inserido depois
Texto sobrescrito
Usamos a tag sup, que mostra um texto sobre escrito, como no expoente de um número
Texto subescrito
Usamos a tag sub, que mostra um texto subescrito, como na fórmula da água, h2O, ou num logaritimo
Fonte monoespaçada
Usamos a tag code, que deixa a visualização melhor, muito usada para mostrar um código fonte
Pré formato
Usamos a tag pre, que deixa o texto como foi digitado, deixando espaços, e tabs também
Citações sem deslocamento
Usamos a tag q, que bota aspas para indicar citação
Citação com deslocamento
Usamos a tag blockquote, usados para citações mais longas, que precisam de um parágrafo só para si. Podemos também colocar um link para o texto original, colocando o parametro cite dentro da tag
Abreviações
Usamos a tag abbr, que permite colocar o significado da abreviação dentro do parametro title, onde quando se passa o mouse por cima aparece esse significado
Texto invertido
Usamos a tag bdo dir="rtl" ou "ltr", bdo significa bi directional override, e a parametro dir, indica a direção de onde o texto vai começar, onde rtl(direita pra esquerda) ltr(esquerda pra direita)

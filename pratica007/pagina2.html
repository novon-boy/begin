Direitos Autorais

Devo tomar muito cuidado com os direitos autorais da imagem pois isso pode prejudicar o meu site, para encontrar imagens sem direitos autorais eu posso seguir os seguintes passos no google imagens:

  1. Tools
  2. Usage rights
  3. Creative common licenses(permite o uso e mudanças)

Sites para encontrar imagens sem direitos autorais:

  • UnSplash
  • Pexels
  • FreePik
  • Rawpixel
  • Pixabay
  • Libreshot
  • Wikimedia Commons

Formato de imagens

JPEG
Ótima compactação, e tamanho muito reduzido
PNG
Capacidade de configurar a opacidade de cada píxel(deixar transparente ou com transparência limitada, é compactado, mas não tão bem quanto o JPEG

Tamanho das imagens

Isso importa muito pois se uma imagem é muito grande, o arquivo é ainda maior, ou seja, devemos evitar usar esse tipo de imagem pois irá deixar o site lento. Se quisermos deixar uma imagem pequena, devemos editar a imagem num editor, e não nas CSS pois fazendo assim o tamanho do arquivo continuará o mesmo, diferente de quando se muda com um editor.

Um site pesado e lento é penalizado pelos mecanismos de pesquisa, onde são retirados das primeiras páginas de busca, o que prejudica MUITO.

Carregar imagem em HTML

Para carregar uma imagem em HTML, usamos "<img src="" alt="">"

src=""
É onde o caminho(caso queira por uma imagem local) ou URL(caso queira por uma imagem externa) está
alt=""
Texto alternativo caso a imagem não carregue, o que ajuda pessoas com deficiência por exemplo

Listas

Listas ordenadas

As listas ordenadas são feitas com a tag ol, onde para cada item na lista usamos li.
Veja o exemplo

        
            <ol>
                <li>Item 1</li>
            </ol>

Listas não ordenadas

Para fazer listas não ordenadas usamos a tag ul, e fazemos a mesma coisa que a lista ordenada para por os itens, usamos a tag li

Listas misturadas

É possível misturar as listas ordenadas com as listas não ordenadas, exemplo;

        
            <ol>
                <li>Item 1</li>
                    <ul>
                        <li>sub item 1<li/>
                    </li>
            </ol>

Tendo como resultado;

  1. Item 1
    • sub item 1

Listas de definição

Para fazermos listas de definição usamos a tag dl, para o termo usamos a tag dt, e para a definição usamos dd

        
            <dl>
                <dt>Termo<dt>
                <dd>Definição<dd>
            </dl>

Tendo como resultado:

Termo
Definição

Para misturar item de listas ordenadas, ou não ordenadas com definições, usamos CSS, podemos fazer isso, ou editar a lista de definição com CSS também, para deixar como uma lista ordenada ou não ordenada

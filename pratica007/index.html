Títulos e parágrafos

    Títulos
    Para escrever um título em HTML usamos as tags h1, para escrever um sub-título menos usamos h2, e assim por diante até o h6; cada h.. está dentro de outro(menos o h1 que é o título principal), por exemplo, um h2, sempre estará dentro de um h1, um h3 sempre estará dentro de um h2 e de um h3, etc. A ordem dos títulos é algo que deve ser seguida corretamente.
    Paragráfos
    Os paragráfos são escritos com a tag p, e estão dentro de títulos/sub-títulos

Quebras de linha

Para quebrar a linha devemos usar a tag br, onde ela pula a linha que estava sendo utilizada antes, servindo em várias situações, como em paragráfos, imagens, etc.

Símbolos

Há muitos símbolos que podem ser usados no HTML 5, eles são chamados de HTML Entities, todos possuem ponto e vírgula no final, e &, exemplos

  • lt - usado para mostrar o sinal de menor <;
  • gt - usado para mostrar o sinal de maior >;
  • nbsp - espaço em branco " "
  • reg - marca registrada ®
  • copy - copyright ©
  • trade - trade mark ™
  • euro - Euro €
  • pound - Libra £
  • yen - Ienes ¥
  • cent - Cents ¢
  • empty - vazio ∅
  • sum - soma ∑
  • delta - delta δ
  • larr - seta esquerda ←
  • uarr - seta cima ↑
  • rarr - seta direita →
  • darr - seta baixo ↓

Emojis

Para usar um emoji no HTML 5, devemos escrever &#x + unicode do emoji, em alguns servidores irá funcionar caso você coloque o emoji direto no código, mas outros não irão, então o certo é fazer o código

Âncoras

Para criar uma âncora usamos a tag <a href="" hrefland="" target="">texto</a>

herf=""
É onde estára o caminho ou URL para a âncora
hreflang=""
Permite indicar qual é o idioma principal do site que estamos ancorando/ligando
Target=""
Atributo que permite a opção de abrir o site da ancora em outra aba, possui os seguintes valores
_blank
Vai abrir o link em uma nova janela em branco
_self
Vai abrir o link na janela ou frame atual(padrão)
_top
Vai desfazer todos os frames e abrir o destino no navegador completo
_parent
Similar ao uso do _top em uma referência a janela mãe
nome-do-frame
Caso esteja usando frames indica o nome da janela ao abrir

Esse link é seu ou dos outros?

Existe um recurso bem interessante para link que é indicar qual é a natureza do destino usando o atributo rel, esse atributo aceita vários valores, entre eles:

next
Indica que o link é a próxima parte do documento atual
prev
Indica que o link é a parte anterior do documento atual
author
indica que o link é para o site do autor do artigo atual
external
Indica que é um link para outro site que não faz parte do site atual
nofollow
Indica que o link é para um site não endossado, como um link pago

Links para download

Para fazer um link para download de arquivo em PDF ou ZIP, por exemplo, usamos a seguinte tag:

        
            <a href="caminhodoarquivol.pdf" download="arquivo1.pdf" type="application/pdf">mensagem</a>
